# MSBAi Online Curriculum Structure

**Launch:** Fall 2026
**Total Duration:** 4 semesters + 1 summer (18 months)
**Total Credits:** 36
**Key Design:** 8-week courses within 16-week semesters; some courses straddle both halves
**Common Thread:** Python + Jupyter Notebooks across all courses
